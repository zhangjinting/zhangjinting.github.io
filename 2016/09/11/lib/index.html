插件

发表于 2016-09-11 | 分类于 Javascript
整理一些好用的插件（持续更新）

插件

MUI

一款前端高性能框架，为解决HTML5在低端Android机上的性能缺陷，mui引入了原生加速，其中最关键的就是webview控件，因此mui若要发挥其全部能力，需和5+ App配合适用，若脱离5+ App，mui功能会受限。
可实现功能：轮播、折叠面板、消息框、滑块、下拉刷新上拉加载等
链接：http://dev.dcloud.net.cn/mui/

Swiper

纯javascript打造的滑动特效插件，面向手机、平板电脑等移动终端。
链接：http://www.swiper.com.cn/

Font Awesome

一款好用的字体图标
链接：http://www.fontawesome.com.cn/

zepto.js

提供jQuery的类似的API，但并不是100%覆盖jQuery，Zepto的一些可选功能是专门针对移动端浏览器的
链接：http://www.css88.com/doc/zeptojs_api/
链接：http://zeptojs.com/

iscroll

移动端滚动插件
链接：http://www.mamicode.com/info-detail-331827.html

fullpage

基于jQuery的插件,它能够很方便、很轻松的制作出全屏网站
链接：http://fullpage.81hu.com/
链接：http://www.bootcdn.cn/fullPage.js/

fastclick

处理移动端click的300毫秒延迟
链接：http://amazeui.org/1.x/javascript/fastclick/

html5shiv

html5shiv使得传统浏览器支持部分HTML5元素并提供相应的基本样式。
链接：http://www.bootcdn.cn/html5shiv/

respond

respond.js解决IE6~8的响应式布局问题
链接：http://www.bootcdn.cn/respond.js/

simplemde

SimpleMDE是一个简单的、可嵌入和美丽的 JS markdown 编辑器。
链接：https://simplemde.com/
链接：https://segmentfault.com/a/1190000009469890

Bulma

Bulma 是一个基于 Flexbox 的现代化的 CSS 框架，设计的初衷就是移动优先（Mobile First），模块化设计，可以轻松用来实现各种简单或者复制的内容布局，浏览器支持：浏览器支持：Chrome、Edge、Firefox、Internet Explorer (10+)、Opera 以及 Safari。
链接：http://www.bootcdn.cn/bulma/

JSON2

是处理IE6 7低版本浏览器json数据兼容问题，序列号和反序列化，json.parse和json.stringify这两个方法在ie8及以上浏览器都能够使用。
链接：http://www.cnblogs.com/youring2/archive/2013/03/01/2938850.html
